Este proyecto está configurado para utilizar la biblioteca OpenCV (en su versión 
3.1 en las configuraciones para Windows, o en cualquier versión que se encuentre
instalada en las configuraciones para GNU/Linux).

El código de ejemplo del main fue tomado de https://gist.github.com/EyalAr/3940636

Al final de la barra de herramientas encontrará una ícono adicional para acceder
a la referencia en línea de la biblioteca.
